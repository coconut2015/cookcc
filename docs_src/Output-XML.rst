XML
===

The output from this language is simply an XML input (``*.xcc`` format)
for CookCC. This output is both serving as a debugging function and a
conversion tool when other input formats to CookCC is available.

XML output can be selected using the ``-lang xml`` command line option.

Command Line Options
--------------------

+----------------+---------+----------------------------------------------------+
| Option         | Version | Description                                        |
|                |         |                                                    |
+================+=========+====================================================+
| ``-o <file>``  | 0.3+    | Specify the complete path for the output file. If  |
|                |         | this option is not specified, the default output   |
|                |         | is the console.                                    |
+----------------+---------+----------------------------------------------------+
